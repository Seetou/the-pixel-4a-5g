Questions fréquentes

Qu'est-ce qu'un téléphone débloqué ?

Un téléphone débloqué est un téléphone qui n'est pas lié à un opérateur spécifique. Quand vous achetez un téléphone Pixel débloqué, vous pouvez choisir l'opérateur et le forfait adaptés à vos besoins. Sur le Google Store, la plupart des téléphones sont débloqués. Important : Les téléphones Pixel fonctionnent avec tous les grands opérateurs. Toutefois, tous les Pixel 4a (5G) et versions ultérieures ne fonctionnent pas sur tous les réseaux 5G.

Qu'est-ce que la technologie eSIM ?

La technologie eSIM vous permet d'ajouter un forfait mobile sur votre Pixel 4a (5G) sans utiliser de carte SIM physique. Quand vous le recevez, votre Pixel4a (5G) est prêt à être activé via la technologie eSIM. Lors de la configuration, vous devez être connecté à un réseau Wi-Fi.

Que contient la boîte du Pixel 4a (5G) ?

Le Pixel 4a (5G) est livré avec un câble USB-C® vers USB-C® (USB® 2.0) d'un mètre, un guide de démarrage rapide, un adaptateur Quick Switch (pour le transfert de données) et un outil pour la carte SIM. Le chargeur 30W USB-C n'est pas fourni avec le Pixel 4a (5G), mais il peut être acheté séparément.
